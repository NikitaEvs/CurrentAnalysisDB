# Семестровый проект по предмету "Базы данных"
---

## Предметная область
В проекте реализована база данных системы анализа потребления в электрической системе.

Основные сущности:
* Измерение производства
* Датчик
* Сетевая компания
* Группа пользователей
* Потребитель электричества
* Солнечная палень
* Пользователь
* Тариф
